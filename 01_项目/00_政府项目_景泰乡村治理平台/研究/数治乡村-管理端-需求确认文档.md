## 1. 系统是做什么的

**一句话说明**：数治乡村管理端是面向网格员/村委工作人员的数字化治理工作台，用于处理随手拍工单、管理考勤、发布信息、管理培训等乡村治理工作。

**谁在使用**：
- **网格员**：处理工单派单、查看考勤统计、处理异常打卡初审、管理培训报名、发布信息
- **村委领导/乡镇领导**：审批异常考勤、终审曝光台、发布预警信息、查看数据统计

### 6.4 状态流转说明

**异常考勤审批流程：**
```
待处理(pending) → 网格员提交 → 待审批(submitted) → 领导审批 → 已通过(approved) / 已驳回(rejected)
```

**曝光台处理流程：**
```
待处理(pending) → 处理中(processing) → 已整改(resolved) / 已关闭(closed)
```

**预警信息状态：**
```
草稿(draft) → 已发布(published) → 已过期(expired)
```

**培训活动状态：**
```
即将开始(upcoming) → 进行中(ongoing) → 已结束(completed)
```

**积分兑换订单状态：**
```
待审核(pending) → 管理员审核 → 待领取(approved) / 已驳回(rejected)
待领取(approved) → 确认领取 → 已完成(completed)
已驳回(rejected) → 积分自动退还至用户账户
```

**商品状态：**
```
已上架(on) ↔ 已下架(off)
```

**账号注销流程：**
```
发起注销 → 短信验证 → 二次确认 → 账号注销完成 → 跳转登录页
```
