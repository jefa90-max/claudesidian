## 项目目标

> [!abstract] 产品定位
> 一款基于自有视频内容的**在线语言学习平台**，通过双语字幕、激活码注册、智能复习路径，帮助用户高效学习语言（仅支持中英双语视频）。

### 核心差异化

| 特点       | 描述                    |
| -------- | --------------------- |
| 内容自主可控   | 管理员后台上传精选视频 + 人工校对字幕  |
| 极简学习体验   | 零学习成本，3步快速上手          |
| 智能复习闭环   | 从观看到记忆的完整学习路径（MVP以收藏回看为主） |

### 目标用户

- **主要用户**：语言学习者（中英双语）
- **次要用户**：专业领域学习者（需要特定领域中英双语视频内容）

### 主要目标

- [ ] 目标 1：完成 MVP 核心功能开发（视频播放 + 双语字幕 + 生词收藏）
- [ ] 目标 2：实现用户激活码注册与学习进度追踪系统
- [ ] 目标 3：搭建管理员后台（视频上传 + 用户管理）
- [ ] 目标 4：上线并获取首批用户反馈

### 成功标准

| 指标       | 目标值      |
| -------- | -------- |
| 视频首次加载   | < 3 秒    |
| 字幕响应延迟   | < 100 ms |
| 词典查询响应   | < 500 ms |
| 页面切换流畅度  | 60 fps   |
| 浏览器兼容    | Chrome、Edge、Safari、Firefox（最近2个版本） |
| 设备响应式    | PC、平板、手机 |

## 功能范围

### P0 - 核心 MVP（第一阶段）

| 模块     | 功能                              |
| ------ | ------------------------------- |
| 视频播放器  | MP4播放 + 快捷键（句维度：←→切句、S单句循环）     |
| 双语字幕   | WebVTT、点击跳转、单句循环、逐句暂停、显示控制、关键词高亮 |
| 字幕列表   | 同步滚动、高亮当前句、点击跳转                 |
| 用户系统   | 激活码注册（一次性码）、登录                  |
| 生词收藏   | 词/短语/句级收藏、回跳原句                  |
| 学习进度   | 到秒续播、继续学习入口                     |
| 视频库    | 列表页 + 详情页                       |
| 管理后台   | 视频上传管理（草稿/已发布/下架）               |

### 前台页面

```
├── 首页（含「继续学习」区块）
```
