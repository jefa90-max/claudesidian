1.切换小程序用户
`packages/api/scripts/reset-wechat-binding.ts`,在 `packages/api` 目录下跑:

```bash
# 列出当前所有绑了真实微信号的用户（dry-run，安全，不改库）
pnpm exec tsx --env-file=.env scripts/reset-wechat-binding.ts

# 一键清掉所有绑定（最常用，重置测试环境）
pnpm exec tsx --env-file=.env scripts/reset-wechat-binding.ts --all --reset

# 只清某个 openid（开发者工具日志里能看到）
pnpm exec tsx --env-file=.env scripts/reset-wechat-binding.ts --openid=oL6yg7ZC04_xxx --reset

# 只清某个用户 id
pnpm exec tsx --env-file=.env scripts/reset-wechat-binding.ts --id=<uuid> --reset
```
cd packages/api
pnpm exec tsx --env-file=.env scripts/reset-wechat-binding.ts --all --reset
