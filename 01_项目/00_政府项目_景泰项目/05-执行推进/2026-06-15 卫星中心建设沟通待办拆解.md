# 2026-06-15 卫星中心建设沟通待办拆解

来源：[[../06-会议纪要/2026-06-15 卫星中心软硬件采购与数据管理项目沟通转录]]

## 待办清单

### P0 - 下周汇报前必须补齐
- [ ] 要求技术方提供卫星中心一期建设的软硬件采购清单。  
  说明：需明确哪些软件必须采购、哪些硬件必须配置、各项费用及配置依据。
- [ ] 要求技术方明确已有模型能力清单及费用边界。  
  说明：区分“已包含在采购费用内的通用模型”和“需另行训练/定制/提升精度的模型”。
- [ ] 要求技术方提供模型定制报价与周期。  
  说明：按地物类别/作物类别列出 15–30 万区间内的估算依据、样本要求和训练周期。
- [ ] 要求技术方提供后续技术支持与运维费用测算。  
  说明：明确首年免费服务范围，第二年起按采购费用 15%/年测算，并给出多年预算口径。
- [ ] 要求技术方整理可用于党政/初审汇报的建设方案材料。  
  说明：材料需服务于建设方案申报、采购计划、人员配置和预算管理。

### P1 - 业务落地方案
- [ ] 选定一个具体业务场景做演示或样例方案。  
  建议场景：违法图斑提取、耕地/永久基本农田范围内疑似问题识别、撂荒识别、经济作物/灌木/果树识别。
- [ ] 梳理一期业务流程：影像自动化处理 → 智能解译 → 人工后处理 → 分发/展示。  
  说明：标明每个环节需要多少人、什么专业能力、哪些环节可自动化。
- [ ] 明确人工后处理工作量测算口径。  
  参考：成都市季度地类处理在自动解译后约需 4 人 1 个月后处理。
- [ ] 结合现有协同单位软件能力，判断无人机处理软件是否需要重复采购。  
  说明：若已有航测/无人机处理软件且格式兼容，可只补齐缺口。

### P2 - 人员与培训
- [ ] 制定卫星中心人员配置建议。  
  说明：区分行政管理人员、专业技术人员、实习生、后处理人员。
- [ ] 明确适合招聘/实习的学校与专业方向。  
  方向可包括遥感、测绘、地理信息、城乡规划、计算机/数据处理等。
- [ ] 制定人员培训计划。  
  说明：包括培训内容、培训周期、上岗标准、是否到技术方现场培训。
- [ ] 评估第一年使用实习生的可行性和成本。  
  说明：可按天计费，后续再逐步培养长期人员。

### P3 - 数据源与亮点项目
- [ ] 要求技术方列出商业数据源采购价格、周期和更新频率。  
  说明：尤其关注两个月、三个月、半年更新一次是否满足业务要求。
- [ ] 要求技术方提供三卫星数据适用场景与成本说明。  
  说明：重点看地质灾害监测、地形检测等是否适合纳入后续规划。
- [ ] 请技术方提供全国自然资源系统可参考案例。  
  说明：关注浙江、上海等地在规划、风貌、耕地保护、城市治理等方向的先进做法。
- [ ] 形成遂宁市可申报的亮点项目清单。  
  方向：城市风貌监测、核心商圈招牌/广告治理、历史遗址保护、耕地保护、规划实施监管、智慧城市支撑等。

## 建议下一步

- 下周三/周四前完成技术方资料收集。
- 将“软硬件采购清单 + 费用测算 + 人员配置 + 运维测算”整理为一页预算口径。
- 将“业务场景 + 流程 + 精度 + 人工后处理”整理为一页业务落地说明。
- 将“亮点项目方向”单独整理成汇报材料，突出遂宁市差异化价值。
